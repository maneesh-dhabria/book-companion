Ch.3 Habit Formation ▾
3/14
Original
Summary

Key Takeaway: Habits are driven by a three-part neurological loop stored in the basal ganglia.

The brain converts repeated behavioral sequences into automatic routines through "chunking." The habit loop:

  • Cue — trigger that initiates behavior
  • Routine — the behavior itself
  • Reward — benefit reinforcing the loop

Once established, the basal ganglia encodes the full loop, reducing prefrontal cortex involvement. This means habits persist even when conscious attention is elsewhere.

The key finding: habit formation is not about willpower but about understanding and redesigning the loop components.

Mobile Views — Design Summary

  • Bottom tab bar: 5 tabs (Library, Search, Concepts, Annotations, Settings) shown consistently across main screens. Reader and detail views hide the tab bar in favor of action bars.
  • Library: Scrollable status filter tabs, compact sort/tag dropdowns, list view with cover thumbnail + title + author + status badge. "+" button for upload.
  • Reader: Back arrow + chapter dropdown in header with position indicator. Segmented Original/Summary toggle. Full-width content. Bottom action bar: Aa (settings), AI chat, annotations, more.
  • Bottom sheets: Annotations and AI Chat use draggable bottom sheets with handle. Dimmed reader content visible above. Sheets can be swiped to dismiss.
  • Search: Full-screen takeover with search input at top, back button, grouped results by book. Same type badges (section/concept/summary/annotation) as desktop.
  • Upload: Full-width 3-step wizard with progress dots. Large drag-drop zone, stacked form fields, full-width Continue button.
  • Concepts: Single-column list (no split panel). Tap concept row to push to detail screen (right chevron hint). Book/chapter tags below each concept.
  • Reader Settings: Full-screen modal. Preset cards in 2-column grid. Font buttons wrap. Sliders for size/spacing. Background color swatches. Live preview at bottom.
  • Processing: Minimized progress bar at top of any screen (tap to expand). Expanded view shows book info, overall progress bar, per-section status list with spinner on active section.
