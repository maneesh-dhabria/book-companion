Set Up Mobile
  • Config toggle: network.allow_lan: true — binds to 0.0.0.0 instead of 127.0.0.1
  • mDNS discovery: Optionally advertise as bookcompanion.local via Bonjour/Avahi — no IP memorization

  • Optional token auth: network.access_token: "my-secret" — appended as query param or entered once on device
  • LAN indicator: When LAN mode is active, header shows a subtle icon + device count

Multi-Device — Design Summary

Read on Your Phone (M7)

  • "Read on your phone": Friendly label replaces "Allow LAN Access". Description: "Your library will be readable from any device on your home network."
  • Mobile Setup Kit: One-action button that enables LAN, shows QR code, and pre-caches recent books for fast mobile reading.
  • Config toggle: network.allow_lan: true binds to 0.0.0.0
  • mDNS: Optional bookcompanion.local discovery via Bonjour
  • QR code: Settings page shows QR for easy phone pairing
  • Token auth: Optional network.access_token for security

Cross-Device Reading Position (S4)

  • Continue where you left off: When connecting from a new device, a banner shows "You were reading [Book Title], [Section Name]" with a Continue button.
  • Auto-sync: Reading position syncs automatically across all connected devices.

Responsive Breakpoints

  • Desktop (≥1024px): Full layout — sidebar icon rail + two/three panel views
  • Tablet (768–1023px): Sidebar collapses to icons only, panels stack as needed
  • Mobile (<768px): Single column, bottom tab navigation, bottom sheets for annotations/AI/settings

Mobile Adaptations

  • Sidebar → Bottom tab bar: Library, Search, Concepts, Notes, Settings
  • Icon rail hidden: Bottom tabs replace it entirely
  • Library defaults to list view: More space-efficient than grid on small screens
  • Book detail: Single column reader. Breadcrumb becomes back arrow + chapter dropdown.
  • Context sidebar → Bottom sheets: Annotations and AI Chat open as draggable bottom sheets (swipe up to expand, swipe down to dismiss)
  • Action bar at bottom: Aa settings, AI button, annotations toggle, more menu — all thumb-reachable
  • Text selection: Long-press triggers the floating toolbar (Highlight, Note, Ask AI, Copy)
  • Upload: Same wizard, full-width steps. File picker uses native mobile file selector.
  • Filters: Dropdowns open as full-screen modal on mobile for easier touch selection.
