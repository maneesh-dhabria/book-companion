Design Updates Summary

  • Header bar: Added dedicated ✦ AI button — always visible, even in focus mode. Separate from 💬 (annotations).
  • Reader settings: Presets at top (system + user-created). Quick buttons for common values + stepper (−/+) for exact values. Font picker with Serif/Sans/Mono shortcuts + searchable "More" dropdown. Background themes: Light/Sepia/Dark/OLED.
  • AI threads: Per-book, multi-thread. Thread list shows title + message count + last date + preview. Inside a thread: chat-style messages with selected text context blocks. AI can reference multiple sections. Responses can be saved as annotations or copied.
  • Multi-section context (R16): "+ Context" pill next to chat input opens a section picker dropdown with checkboxes. Selected sections appear as dismissible chips above the input. AI uses all checked sections as context for responses.
  • "+ Save Current": Saves current reader settings as a named preset for reuse.
  • Save-as-note toast (P7): After saving an AI response as a note, a green confirmation toast appears with "Saved as annotation on Chapter N" and an "Open" link to navigate to it.
  • AI response progress bar (P12): A thin 2px animated indeterminate progress bar appears below AI responses during streaming, using the accent color.
