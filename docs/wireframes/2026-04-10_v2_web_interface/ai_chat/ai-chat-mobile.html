Can you explain how attention residue affects daily productivity?

Attention residue works like this:

1. Task switch -- You move from Task A to Task B

2. Residue forms -- Part of your mind stays on Task A

3. Performance drops -- Task B gets reduced cognitive capacity

Newport cites Sophie Leroy's research: even brief checks of email or Slack create residue that lasts 10-20 minutes.

AI Chat Mobile -- Adaptations

  • Bottom sheet: AI chat opens as a draggable bottom sheet over dimmed reader content. Swipe down to minimize.
  • Thread list view: Shows all chat threads for the current book. Each thread card shows title, first message preview, chapter context, and message count.
  • Active thread view: Back arrow navigates to thread list. Messages in chat bubble style. AI typing indicator shown as dots.
  • Selected text context: When user selects text in reader and opens AI chat, the selected text appears as a context block above the conversation.
  • Chat input at bottom: Fixed input bar with send button. Auto-focuses keyboard when sheet opens.
  • Multi-section context (R16): "+" pill next to chat input opens a section picker as a full bottom sheet with checkboxes and page counts. Selected sections appear as dismissible chips above the input. Swipe down to dismiss the picker.
  • Thread navigation: Back arrow in thread header returns to thread list. "+ New" creates a fresh thread.
