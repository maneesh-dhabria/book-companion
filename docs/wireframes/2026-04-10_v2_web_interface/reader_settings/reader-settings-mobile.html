Reader Settings Mobile -- Adaptations

  • Full-screen modal: Covers entire screen (not a popover). "Done" button dismisses back to reader.
  • Preset cards: 2-column grid. Each card shows name + size/spacing summary. Tap to apply all settings at once.
  • Font: 4 primary fonts shown (Georgia, Merriweather, Inter, Fira Code). "All fonts" link expands to reveal 4 more (Literata, Lora, Open Sans, JetBrains). Buttons wrap on narrow screens.
  • Named presets for size/spacing: Slider labels show current numeric value (16px, 1.6). Presets set all at once.
  • Background swatches: Row of circular swatches. Active swatch has accent ring. White, sepia, gray, dark, black options.
  • Live preview: Preview block at bottom updates in real-time as user adjusts settings.
  • Content width locked: On mobile, content width slider is maxed (full width) since there is no horizontal space to constrain.
