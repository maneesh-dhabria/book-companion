Reader Settings — Final Summary

  • Presets: Visual square cards showing background color + font "Aa" rendering + font name + spacing value. Active preset has accent border + check. User presets show edit icon. "+ Save Current" to create new.
  • Font: 4 primary fonts shown by default (Georgia, Merriweather, Inter, Fira Code). "All fonts ▸" link expands to reveal 4 additional fonts (Literata, Lora, Open Sans, JetBrains). Reduces initial cognitive load while keeping full selection accessible.
  • Size: Named buttons (Small / Medium / Large / XL) + manual stepper (−/+ with px value).
  • Line Spacing: Named buttons (Tight / Normal / Relaxed / Spacious) + manual stepper.
  • Content Width: Named buttons (Narrow / Medium / Wide / Full) + manual stepper (px).
  • Background: Preset swatches (Light / Sepia / Dark / OLED / Dracula) + 🎨 Custom button → opens full color picker with gradient area, hue slider, and hex input. Text color auto-adjusts for contrast.
  • Live preview: Sample text at bottom updates in real-time with current settings. Shows font + size + spacing + width + background summary below.
