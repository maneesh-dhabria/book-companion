Mobile App Shell -- Adaptations

  • Bottom tab bar: 5 tabs (Library, Search, Concepts, Annotations, Settings) replace desktop sidebar navigation. Persistent on all main screens.
  • Header varies by context: Main tabs show hamburger + title; Search tab shows full-width search input; Detail views show back arrow + title.
  • No desktop sidebar: Navigation is entirely via bottom tabs. Hamburger menu provides overflow items (profile, help, import/export).
  • Detail views hide tabs: Reader, concept detail, and annotation detail replace the bottom tab bar with a contextual action bar.
  • Search accessible via tab: Tapping Search tab activates full-screen search with auto-focused input field.
