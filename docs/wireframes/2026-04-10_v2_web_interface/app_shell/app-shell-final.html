Shell Component Reference
Icon Rail Sidebar (56px wide)
  • Fixed left column, always visible
  • Logo at top, Settings pinned to bottom
  • Active item: filled accent background, white icon
  • Inactive items: 40% opacity, no background
  • 4 navigation targets: Library, Concepts, Annotations, Settings
Top Bar (48px tall)
  • Page title area: set by the active page
  • Search input: persistent, opens ⌘K command palette on click/shortcut
  • + Upload button: global action, always accessible
  • Remains consistent across all pages
Content Area
  • Fills remaining space (flex:1)
  • Each page manages its own sub-layout
  • Pages may add filter bars, split panes, etc.
  • Scrolling is scoped to this area
Navigation States
  • Only one nav item active at a time
  • Click switches page content below top bar
  • URL updates for deep linking
  • Tooltips show labels on hover
