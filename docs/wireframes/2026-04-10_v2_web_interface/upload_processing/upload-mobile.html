Upload Mobile -- Adaptations

  • Compact step indicator: 3 numbered dots (matching desktop: Upload, Configure, Process) with connecting lines. Slightly smaller (22px) to fit at 320px. Completed steps show green checkmark. Metadata, Structure, and Preset are sub-steps within Configure.
  • Large drag-drop zone: Adapted for touch with prominent "Browse Files" button. Works with native iOS/Android file pickers.
  • Stacked form fields: All form fields are full-width and vertically stacked. No side-by-side fields at mobile width.
  • Full-width CTA: "Continue" / "Import Book" button spans full width at bottom, fixed above keyboard when input is focused.
  • Preset selection: 2-column grid for summarization presets, same pattern as reader settings.
  • Toggle switches: iOS-style toggle switches for boolean options (auto-summarize, eval, concepts).
