Refinements Summary

  • Duplicate detection (same hash): Warning dialog with 3 options — go to existing book, re-import (marks existing data stale), or cancel.
  • Different format detection: Side-by-side comparison of existing vs uploading. Options — add as new book, replace existing, or cancel.
  • Eval visibility on presets: Expanded preset cards show assertions grouped by category (Faithfulness/Completeness/Coherence/Specificity/Format) with active (green) and skipped (orange strikethrough) badges. Explanation of why each assertion is skipped. Auto-skip rules explained.
  • Preset management in Settings: Full CRUD page under Settings → Summarization Presets. List/detail layout. System presets are read-only with "Duplicate as User Preset" action. User presets fully editable with facet selectors + skip_assertions toggles.
  • Upload wizard links to Settings: "Manage Presets →" link. "+ Create New" in wizard opens a creation modal (same interface, doesn't navigate away).
