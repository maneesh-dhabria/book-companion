Upload & Processing — Complete Summary

  • 5-step wizard: Upload → Metadata → Structure Review → Preset Selection → Processing
  • Preset details: Expandable cards showing 4 facets + prompt preview + skipped assertions
  • Custom preset: 2×2 grid of facet selectors (Style, Audience, Compression, Focus), each with card-style options showing name + description. Option to save as reusable preset.
  • Merge: Checkbox selection on section list → "Merge Selected" toolbar → preview with combined title + char count → confirm
  • Split: Choose section → toggle between "At heading" (auto-detected) or "Manual position" → preview Part A/B with titles + sizes → confirm
  • Minimizable progress: Full progress card in Book Detail (expandable). Minimize to persistent bottom bar showing book name + progress. Supports multiple concurrent jobs stacked. "Expand" to return to full view. ✕ to dismiss (job continues in background).
  • Undo toast: After merge or split, a dark floating toast appears at bottom-center with "Undo" link. Auto-dismisses after 10 seconds with a shrinking progress bar indicator.
  • Recommended preset: "Balanced" preset marked with star icon + "Recommended" badge and pre-selected (accent border + highlight background).
  • Cost/time estimate: Informational card below processing options showing estimated time, API cost, and section count. Muted note about enabling in Settings.
  • Auto-skip structure review: When all sections are well-structured, a green banner with checkmark replaces warnings. Section table shown at reduced prominence. Prominent "Next" button in the banner.
