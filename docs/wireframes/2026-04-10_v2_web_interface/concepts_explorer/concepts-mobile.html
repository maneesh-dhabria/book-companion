Definition
Attention residue refers to the cognitive carry-over that occurs when switching between tasks. Part of your mental capacity remains focused on the previous task, reducing performance on the current one. The effect persists even after brief task switches like checking email.

Concepts Explorer Mobile -- Adaptations

  • Single column list: No split panel layout. Full-width concept cards with brief description and book/chapter tags.
  • Tap-to-push navigation: Right chevron on each row hints at detail view. Tapping pushes to a new screen with back button.
  • Detail view sections: Definition, source book link, key quotes, related concepts, and user notes/annotations.
  • Related concepts are tappable: Each related concept links to its own detail view (same push navigation).
  • Source book link: Tapping the source card navigates to the reader at the relevant chapter.
  • Filter chips: Book filter chips at top of list view. Scrollable if many books in library.
