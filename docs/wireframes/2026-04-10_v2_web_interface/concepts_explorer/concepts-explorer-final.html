Attention Residue
From: Deep Work · Cal Newport
DEFINITION Edit
The cognitive carry-over effect that occurs when switching between tasks. When you move from Task A to Task B, a portion of your attention remains "stuck" on Task A, reducing your cognitive performance on Task B. This effect was demonstrated in research by Sophie Leroy.

Annotations & Concepts — Design Summary

  • Annotations page: Header with count + search + export. Filter bar (Book, Type, Tags) + Group by (Book/Tag/Section/None) + Sort (Newest/Oldest/Book order). Cards show: type badge (highlight/note/freeform), source breadcrumb, quoted text, note, tags, link count. Grouped by book with collapsible accordions. Click navigates to source in reader.
  • Concepts Explorer: Two-panel layout — concept list (left, grouped by book) + concept detail (right). List shows term + "edited" badge + section count. Detail shows: definition (editable), section appearances (clickable), related concepts as chips (cross-book concepts in purple), related annotations.
  • Cross-book connections: Related concepts from other books shown with distinct color + ↗ icon. Clicking navigates to that concept in the other book's context.
  • Filtering pattern: Same multi-select dropdown pattern as Library page for consistency (Book, Tags, Type filters).
  • Both pages auto-filter when accessed from Book Detail: If you click "Concepts" from within Deep Work's reader, the explorer opens pre-filtered to Deep Work.
