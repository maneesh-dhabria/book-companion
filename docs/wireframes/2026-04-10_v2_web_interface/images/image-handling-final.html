Ch.3 — The Science of Habit Formation Summary practitioner_bullets preset

Key Takeaway: Habits are driven by a three-part neurological loop stored in the basal ganglia, which operates independently of conscious decision-making.

The brain converts repeated behavioral sequences into automatic routines through a process called "chunking." This habit loop consists of three components:

🖼
Figure 3.1 — The Habit Loop
KEY
The Habit Loop: cue, routine, reward View in original context
  • Cue — the trigger that initiates the behavior
  • Routine — the behavior itself (physical, mental, or emotional)
  • Reward — the benefit that reinforces the loop

Once established, the basal ganglia encodes the full loop, reducing prefrontal cortex involvement. This means habits persist even when conscious attention is directed elsewhere.

Image Handling — Design Summary

  • Relevance badges: KEY (blue), SUPPLEMENTARY (orange), DECORATIVE (gray). Positioned top-right of each image. Controls how images appear in summaries.
  • Original content: All images shown inline with captions and alt text. KEY images are full-width; DECORATIVE images are smaller and slightly faded.
  • Summary view: KEY images are shown inline at the point they are referenced. SUPPLEMENTARY images are collapsed in an "Image References" section at the bottom with thumbnails, captions, and links to original context.
  • Lightbox: Clicking any image opens full-screen overlay with enlarged view, caption, relevance badge, alt text, and prev/next navigation. ESC or close button to dismiss.
  • AI context: Image count shown as context indicator. AI can reference images by figure number. Clickable image cards within AI responses link to the lightbox view.
  • Alt text: Always available via tooltip in reader view and shown in full in lightbox. Passed to AI context for image understanding.
