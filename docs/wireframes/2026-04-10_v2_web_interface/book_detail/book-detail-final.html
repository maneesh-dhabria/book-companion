Open Floor Plans Metric Black Hole Busyness as Proxy
  • Newport identifies three trends making deep work rare: open floor plans, instant messaging culture, and social media presence expectations Highlighted in Original
  • The Metric Black Hole: knowledge work lacks clear productivity metrics, so workers default to visible busyness
  • Busyness as a Proxy for Productivity: without better measures, doing lots of stuff in a visible manner becomes the default Note in Original
  • The cult of the Internet: any technology that offers any benefit is embraced without cost-benefit analysis of its impact on deep work

Book Detail — Design Summary

  • Breadcrumb header: Library › Book Title › Chapter (dropdown). Chapter segment opens full TOC with search + eval status dots.
  • Original/Summary toggle: Segmented control, not tabs. One or the other.
  • Eval: Collapsed to a single trust badge pill (green/yellow/red). "View details" link navigates to a dedicated eval page instead of inline expansion (P5).
  • Right sidebar: 2 tabs — Annotations | Ask AI. Collapsible. Badge on 💬 icon shows count when closed.
  • Concepts: Clickable chips in summary. Click → tooltip with definition + cross-references.
  • Reader settings (Aa): Popover with font size, line spacing, content width, background theme (Light/Sepia/Dark/OLED).
  • Text selection: Floating toolbar → Highlight, Note, Ask AI, Link, Copy.
  • Actions: Summarize + Evaluate accessible from book-level breadcrumb dropdown or book summary page.
