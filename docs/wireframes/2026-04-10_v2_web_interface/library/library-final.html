✓ Library Page — Approved Design

  • App shell: 56px icon rail sidebar (Library, Concepts, Annotations, Settings)
  • Top bar: Page title + persistent search input (⌘K) + Upload button
  • View tabs: Saved filter+sort+display combos with + New View
  • Filter row: Uniform multi-select dropdowns (Tags, Status, Author, Eval, Format) + Sort + View switcher (Grid/List/Table)
  • Book cards: Cover, title, author, tags, processing status + eval score, section count
  • Views save: All filters, sort, display mode, table columns
  • Live filter counts (M5): Each filter dropdown option shows "X books" count next to it. Counts cross-update dynamically — e.g., selecting "Summarized" in Status recalculates counts in Tags to reflect only summarized books. Provides instant feedback on filter impact before applying.
