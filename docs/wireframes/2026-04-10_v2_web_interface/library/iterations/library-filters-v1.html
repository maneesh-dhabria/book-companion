Design Rationale

  • Tier 1 (always visible) — Status segmented control with counts. This is the most-used filter. Sort dropdown + view switcher (grid/list/table) live here too.
  • Tier 2 (on demand) — "+ Filter" opens a popover to pick dimensions: Author, Tag, Eval Status, Format, Date Added. Each dimension opens a searchable sub-menu. Active filters appear as removable chips.
  • Filter logic — AND between dimensions (Author AND Tag), OR within a dimension (Tag: Business OR Psychology). Standard mental model.
  • Result count updates — Header shows "4 of 12 books" when filtered. Active filter summary below chips.
  • Clear all — One click to reset all secondary filters. Status tabs reset independently.
  • Sort options — Recent (default), Title A-Z, Author A-Z, Most sections, Date added
  • View modes — Grid (covers, visual), List (compact rows), Table (all metadata columns)
