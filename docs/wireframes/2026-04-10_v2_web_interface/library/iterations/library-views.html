How it works

  • View tabs — Saved filter+sort+display combos. Switch in 1 click. "All Books" is the default.
  • + New View — Creates a view from current filter state, or from scratch. Name it, save it.
  • ⋯ menu — Edit view (rename, reorder), duplicate, delete, set as default
  • Tag pills (primary) — Always visible, 1-click filtering. Works within any view.
  • Filter dropdowns (secondary) — Status, Author, Eval, Format. Each is a multi-select popover. Active filters show as blue chips with values + ✕ to clear.
  • View remembers everything — Filters, sort order, display mode (grid/list/table), and even visible table columns.
  • Ad-hoc refinement — Change filters within a view without saving. A subtle indicator shows "unsaved changes" with option to save or revert.
