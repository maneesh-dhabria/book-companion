Library Mobile -- Adaptations

  • List view only: No grid/card layout at mobile. Each row: small cover thumbnail (36x52px) + title + author + status badge.
  • Custom view tabs (R5): Horizontal pill bar syncs desktop-created views (All Books / Currently Reading / Business Shelf / Needs Review). Read-only on mobile; create/edit views on desktop.
  • Compact filter row: Sort and Tag dropdowns in a single row below tabs. Tap to open native picker.
  • Title truncation: Long titles use text-overflow:ellipsis. Author name shown below.
  • "+" button: Top-right add button navigates to upload flow.
  • Empty state: Stacked vertically with full-width CTA button. 3-step how-it-works shown as compact horizontal rows.
  • Terminology (R12): Bottom tab bar uses "Annotations" (not "Notes") to match desktop sidebar.
